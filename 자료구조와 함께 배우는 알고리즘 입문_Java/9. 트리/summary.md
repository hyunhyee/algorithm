# 트리
- 데이터의 계층관계를 나타내는 자료구조
- node
- edge
- 루트
- 리프
- 안쪽노드
- 자식
- 부모
- 형제
- 조상
- 자손
- level
- degree
- 높이
- subtree
- null tree

### 순서 트리와 무순서 트리
형제 노드 사이의 순서 관계를 따지는지 아닌지에 따라 ordered tree, unordered tree로 구분


### 순서 트리 탐색
- 너비 우선 탐색(breadth =first search)
- 깊이 우선 탐색(depth-first search)

- 전위 순회(preorder)
- 중위 순회(inorder)
- 후위 순회(postorder)
